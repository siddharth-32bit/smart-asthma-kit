# Smart Asthma Kit

IoT-based wearable asthma monitoring and spirometry system using ESP32.

## Overview

The Smart Asthma Kit is a health monitoring system designed to assist asthma patients through continuous monitoring of respiratory and environmental parameters.

The project consists of:
- Wearable wristband module
- Portable spirometer module
- Web-based monitoring dashboard

The system monitors:
- SpO₂
- Heart rate
- Temperature
- Humidity
- Air quality
- Lung airflow

Real-time sensor data is transmitted using WiFi/BLE communication for monitoring and alert generation.

---

## Hardware Components

### Wristband Module
- XIAO ESP32-C6
- MAX30102
- BME680
- Li-Po Battery
- TP4056 Charging Module

### Spirometer Module
- Arduino UNO R3
- MPX5010DP Pressure Sensor
